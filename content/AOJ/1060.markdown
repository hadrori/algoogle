#### 問題概要

****

LCMがL(<=1,000,000,000,000)になる2整数a,b(a<=b)の組の数

#### 解法

****

<div> $ {\displaystyle

L = p_{0}^{e_{0}}p_{1}^{e_{1}}...p_{k}^{e_{k}}

} $</div>

pは素数とする.  
このときある2数のLCMがLになるには, 各pについて2数のどちらかは `$ p^{e} $`を含む.  
もう片方のpは0からe乗のいずれかになる.  
その組合せを考えれば良い.  

<div> $ {\displaystyle

(1+\prod_{i=0}^{k} (2e_{i}+1))/2

} $</div>
