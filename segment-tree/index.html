Segment Tree

基本情報

計算量 O(log N)
用途 区間に対するクエリの処理

N := 区間の幅

問題

解説

Segment Treeは主に区間に対するクエリを処理するために使われる.
完全二分木で実装されるので各クエリの計算量はO(log N)になる.
自由度が高く, 区間を扱う様々なものに利用される.
コードではRMQの実装.
